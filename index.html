number2words.js

number2words.js is a very simple javascript plugin which change number in words for example:-
if you have a number 123 , that plugin change this to words( "one hundred twenty two").

number2words() :-

number2words() is a core function of that pluglin. it's carry a number as argument.
if you simply call number2words(23), it's return " twenty three".
